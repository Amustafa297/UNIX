TYPES OF UNIX

UNIX comes in a variety of variations, but they all have some things in common. Sun Solaris, GNU/Linux, and MacOS X are the most popular UNIX variants. Linux, on the other hand, is packaged as a Linux distribution. There are a number of Linux distributions available, both free and paid.

SUN SOLARIS

Solaris is the UNIX-based operating system of Sun Microsystems. The first version of SunOS was published in 1982. Field of Application:
- CAD (computer aided design) applications
- Stable system for databases, data centre
- Intranet server as well as Internet or file server, Internet client

System Environment:
- Optional CDE 1.4 or OpenWindows 6.4.1
- SPARC platform and Intel processors, PowerPC
- supports new hardware technologies like USB, FireWire, SCSI, Hot Plug, ACPI
- Scalability: more than of 4 gbyte RAM, max. 64 CPUs
- File system: UFS (0x83), logging of all writing processes, protection against inconsistencies - Read/Write: FAT, FAT32, ISO9660
- Java support, Perl integrated for CGI programming among others

Strength;
- Support LDAP authentication and NDS
- Supports automatic and dynamic reconfiguration of hardware devices
- Cover various Internet and intranet applications like DNA, Senmail, IPv6, IPsec - High Internet Security

GNU/LINUX

The GNU Project began in 1984 with the goal of creating the GNU system, a complete UNIX-like operating system that is free software. GNU/Linux systems are variants of the GNU operating system that use the Linux kernel. Though these systems are commonly referred to as "Linux," they are more appropriately referred to as GNU/Linux systems. Strength:
- Security
- free
- lightweight
- stability - performance
- suitable for programmers.

MACOS X

Mac OS X is a Unix-based graphical operating system based on technologies developed at NeXT (from the second half of the 1980s to Apple's purchase of the company in late 1996), as well as technologies from Apple's "classic" OS and FreeBSD. Every iteration of Mac OS X for Intel processors since its sixth release, Mac OS X 10.5.0 "LeopaArd," has received UNIX 03 certification.
Strength:
Maximum performance on Mac hardware
