Difference between UNIX and Windows

Since most of the people are using and are used to Windows operating system here are some of the main differences between UNIX and Windows.
1. UNIX uses Command Line Interface while Windows uses Graphical User Interface (GUI).
2. UNIX allows multiprocessing while Windows does not.
3. UNIX is command-based operating system and windows is menu based.
4. Although Windows has improved greatly its stability UNIX is still superior in this context.
5. In UNIX users can save two files with the same name while that is not possible in Windows operating systems.
